ASICS V14 PWA

Contenu :
- index.html
- manifest.webmanifest
- sw.js
- icons/

Installation :
1. Héberge ce dossier sur une URL HTTPS (GitHub Pages, Netlify, Cloudflare Pages).
2. Ouvre l'URL dans Chrome ou Samsung Internet.
3. Menu navigateur → Ajouter à l'écran d'accueil / Installer l'application.

Important :
- Une PWA ne s'installe pas proprement depuis un fichier local file://
- Il faut une URL HTTPS ou localhost
